# AngriL DevTools - Developer Efficiency Platform

[![AngriL DevTools](https://www.angril.com)](https://www.angril.com) | [GitHub Repository](https://github.com/netatall/angril-devtools)

## Overview

Free online developer tools platform - JSON formatter, Base64 encoder/decoder, timestamp converter and more.

## Features

• **JSON Formatter** - Real-time formatting and validation
\n• **Base64 Encoder/Decoder** - Bidirectional encoding/decoding
\n• **Timestamp Converter** - Unix timestamp → date time conversion\n• **Advanced Features(Pro Only)** - Batch processing, API access, History recording

## Usage

1. Visit [www.angril.com](https://www.angril.com)
2, Select the tool you need
3. Input your data
4. Get instant results
